# Student Performance Prediction System

This project predicts student academic performance using machine learning.

## Dataset
UCI Student Performance Dataset

## Models Used
- Logistic Regression
- Decision Tree
- Artificial Neural Network (MLP)

## Steps
1. Data preprocessing
2. Feature encoding
3. Train-test split
4. Model training
5. Evaluation and comparison

## Results

| Model | Accuracy |
|------|---------|
| Logistic Regression | 0.38 |
| Decision Tree | 0.34 |
| Neural Network | 0.34 |

## Outcome
The system predicts student grade categories (Fail, C, B, A) and identifies important factors affecting performance.
